# `BlazorWebAssemblyDotnetOnWebWorkers`

This sample accompanies the article [ASP.NET Core Blazor with .NET on Web Workers](https://learn.microsoft.com/aspnet/core/blazor/blazor-on-dotnet-with-web-workers).

# Startup

Windows:

```bash
./run.cmd cleanbuild
./run.cmd run
```

Linux:

```bash
./run.sh cleanbuild
./run.sh run
```
